# Assets da Landing Page

Esta pasta deve conter:

## Obrigatórios

1. **screenshot.png** - Screenshot do aplicativo (1280x800 recomendado)
2. **favicon.png** - Ícone do site (32x32 ou 64x64)
3. **og-image.png** - Imagem para compartilhamento em redes sociais (1200x630)

## Opcionais

- **logo.svg** - Logo vetorial
- **video-demo.mp4** - Vídeo de demonstração

## Como criar os assets

### Screenshot
1. Abra o app com `npm run tauri:dev`
2. Tire um screenshot da tela principal
3. Salve como `screenshot.png`

### Favicon
- Use o ícone existente em `src-tauri/icons/32x32.png`
- Copie para cá como `favicon.png`

### OG Image (para redes sociais)
- Crie uma imagem 1200x630px com:
  - Logo do TikTrend Finder
  - Tagline: "Descubra Produtos Virais"
  - Screenshot do app
